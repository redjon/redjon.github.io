JavaScript를 웹을 위한 스크립트 언어가 아니라 독립적인 프로그래밍 언어로 사용할 수 있다. 맥에서 Terminal을 열고 다음 명령을 내리면 JavaScript 인터랙티브 모드가 실행된다.

```
$ /System/Library/Frameworks/JavaScriptCore.framework/Resources/jsc
```
