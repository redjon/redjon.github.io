참고

- [Jeykll을 사용하여 GitHub Pages 만들기](http://blog.saltfactory.net/jekyll/upgrade-github-pages-dependency-versions.html)



설치

```
$ gem install jekyll
```

사용 방법

```
$ mkdir mysite
$ cd myste
$ jekyll new . --force
$ jekyll serve
```

웹브라우저에서 `localhost:4000`을 연다.



사이트 설정

`_config.yaml`
